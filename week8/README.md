# Week 8: Inform II – Text and Image Data in Practice

## 📚 Overview

This week introduces the principles of working with both text and image data for descriptive analysis. While we will discuss key concepts for both data types, all practical exercises and examples will focus on text data only. You'll learn how to process, summarize, and interpret complex text data, extract meaningful descriptive statistics, and communicate your insights effectively.

## 🎯 Learning Objectives

By the end of this week you will be able to:

- Explain the principles of working with both text and image data for descriptive analysis
- Load, preprocess, and summarize text data in Python
- Apply basic algorithms for text analysis (e.g., word frequency, keyword extraction)
- Generate and interpret descriptive statistics for text datasets
- Use pre-trained language models (LLMs) for text classification
- Communicate insights from text data effectively
