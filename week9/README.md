# Week 9: Guide I – Predicting Outcomes

## 📚 Overview

This week introduces the fundamentals of predictive modeling, with a special focus on image analysis. You will learn how to approach outcome prediction tasks, understand the basics of supervised learning, and apply practical techniques for analyzing and interpreting images.

## 🎯 Learning Objectives

By the end of this week you will be able to:

- Explain the principles of predictive modeling and supervised learning
- Understand the basics of image data and its structure
- Load and preprocess image datasets
- Apply basic image analysis techniques (e.g., visualization, augmentation)
- Train and evaluate a simple image classifier
- Interpret and communicate results from image-based predictions

## 🏁 Mini-Deliverable

Create a reproducible notebook that:

1. Loads and explores an image dataset (e.g., X-rays, Plant pictures)
2. Implements basic image preprocessing and visualization
3. Trains and evaluates a simple image classifier
4. Summarizes and interprets predictive results
5. Documents your workflow and insights
