# Week 10: Guide II – Interpreting Predictions & Understanding Limits

## 📚 Overview

This week you'll learn how to evaluate machine learning models using time-series analysis and understand the limitations of predictions. You'll explore key evaluation metrics (MSE, RMSE, WAPE, MAPE) and learn how k-fold cross-validation helps you build robust, reliable models.

## 🎯 Learning Objectives

By the end of this week, you will be able to:

- Understand time-series data and its unique characteristics
- Calculate and interpret evaluation metrics: MSE, RMSE, WAPE, MAPE
- Implement k-fold cross-validation for model assessment
- Compare model performance across different evaluation metrics
- Understand when each metric is most appropriate for your problem
- Recognize and explain the limits of predictions
