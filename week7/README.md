# Week 7: Inform I – Describing the Current State with Data (Maps in Python Plotly)

## 📚 Overview

This week focuses on using maps and geospatial visualizations to describe the current state of a region, country, or community. You'll learn how to use Python Plotly to create interactive maps, visualize spatial patterns, and communicate insights about the present situation using data.

## 🎯 Learning Objectives

By the end of this week you will be able to:

- Write titles that communicate insights, not labels
- Distinguish observation vs interpretation
- Communicate uncertainty and limitations
- Understand the basics of geospatial data and mapping
- Create interactive maps with Plotly (scatter, choropleth, bubble)

## 🏁 Mini-Deliverable

Create a reproducible notebook that:

1. Loads a geospatial dataset (e.g., country, city, or region data)
2. Visualizes the data on an interactive map using Plotly
3. Customizes map appearance and tooltips
4. Describes spatial patterns and the current state
5. Documents your process and interpretation
